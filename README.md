# 🛡 Cyber Security Telegram Bot

A professional Telegram bot built with Python that provides multiple cybersecurity tools such as password analysis, hash generation, IP lookup, WHOIS lookup, CVE search, VirusTotal scanning, port scanning, cybersecurity news, and PDF report generation.

---

## 🚀 Features

- 🔐 Password Strength Checker
- 🔒 Hash Generator (MD5, SHA1, SHA256)
- 🌍 IP Address Lookup
- 🌐 WHOIS Domain Lookup
- 🛡 CVE Vulnerability Search
- 🔍 VirusTotal URL Scanner
- 🚪 Port Scanner
- 📰 Latest Cybersecurity News
- 📄 PDF Report Generator
- 📊 Admin Statistics
- 📝 Command Logging
- 🎛 Interactive Inline Keyboard

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/CyberSecurity-Telegram-Bot.git
```

Go to the project folder:

```bash
cd CyberSecurity-Telegram-Bot
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the bot:

```bash
python bot.py
```
